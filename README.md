# Del_Bot_message
Телеграм бот позволяет удалять сообщения других ботов. Работает из под личного аккаунта Телеграм. Настроен под удаления сообщений Глиса бота, но можно изменить фильтр слов под свои задачи.

### Как установить
Клонируйте код.
- Скачайте код.
- Установите витруальное окружение командой. `python -m venv venv`
- Запустите виртуальное окружение командой. `venv\Scripts\activate`
- Установите зависимости командой. `pip install -r requirements.txt`
- Возьмине ненужный аккаут Телеграм и с сайта(https://my.telegram.org/) получите api_id и api_hash.
- Создайте файл .env, в нем создайте две переменные api_id и api_hash в них поместите данные из предидущего пункта.
- Запуститу main.py. `python main.py`
- При первом запуске pyrogram запросит номер телефона и подтверждение с него от аккаунта с которого будет работать бот для получения куки.

### Что использует
Бот для работы использует библиотеки: Pyrogram, TgCrypto, python-dotenv.

Код написан для одной себя.
